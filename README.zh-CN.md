# Tab Out

[English](README.md) · **简体中文**

**掌控你的标签页。**

Tab Out 是一款 Chrome 扩展,为你提供一个清爽的仪表盘,展示所有已打开的标签页。点击工具栏图标即可打开——你的新标签页保持不变。标签页按域名分组,主页(Gmail、X、LinkedIn 等)会被单独归类。跨所有标签页搜索、把标签页拖进自定义分组,还能配上清脆的 swoosh 声 + 彩纸特效优雅地关闭它们。

没有服务器。没有账号。没有外部 API 调用。只是一个 Chrome 扩展。

## 功能特性

- **一眼看清所有标签页** 在清爽的网格上,按域名分组
- **点击即开** 点击工具栏图标打开仪表盘;新标签页保持 Chrome 默认行为
- **搜索打开的标签页** 顶部的搜索框把所有标签页过滤成扁平列表,瞬间找到任意页面
- **自定义分组** 创建命名分组,把标签页拖进去,按你的方式整理仪表盘
- **主页分组** 把 Gmail 收件箱、X 首页、YouTube、LinkedIn、GitHub 首页归到同一张卡片
- **优雅地关闭标签页** 配有 swoosh 声 + 彩纸迸发特效
- **重复检测** 当同一个页面被打开两次时会标记出来,一键清理
- **默认安全** 每个关闭/删除操作都会先要求确认
- **点击任意标签页即可跳转** 可跨窗口,不会新开标签页
- **稍后查看** 关闭前把标签页收藏到清单里
- **本地分组** 在每个标签页旁显示端口号,方便区分你的各种本地开发项目
- **可展开分组** 默认显示前 8 个标签页,可点击「还有 N 个」展开
- **100% 本地** 你的数据永远不会离开你的设备
- **纯 Chrome 扩展** 没有服务器、没有 Node.js、没有 npm,加载扩展即用

## 手动安装

**1. 克隆仓库**

```bash
git clone https://github.com/rb05180518/tab-manager.git
```

**2. 加载 Chrome 扩展**

1. 打开 Chrome,进入 `chrome://extensions`
2. 开启右上角的**「开发者模式」**
3. 点击**「加载已解压的扩展程序」**
4. 选择克隆下来的仓库里的 `extension/` 文件夹

**3. 固定并打开它**

1. 点击 Chrome 工具栏的**拼图图标**,把 **Tab Out** 固定下来方便访问。
2. 点击 **Tab Out** 图标打开仪表盘。(你的正常新标签页不受影响。)

## 工作原理

```
你点击 Tab Out 工具栏图标
  -> Tab Out 按域名分组显示你打开的标签页
  -> 顶部搜索框把所有标签页过滤成扁平列表
  -> 创建自定义分组,把标签页拖进去
  -> 主页(Gmail、X 等)有自己的分组置顶
  -> 点击任意标签页标题即可跳转
  -> 关闭用完的分组(swoosh + 彩纸)——会先确认
  -> 关闭前可把标签页加入「稍后查看」
```

所有功能都在 Chrome 扩展内运行。没有外部服务器,没有 API 调用,数据不会发送到任何地方。已保存的标签页和自定义分组都存储在 `chrome.storage.local` 中。
